# rlp-php

RLP (Recursive Length Prefix) encoder and decoder — the canonical serialization format used throughout Ethereum (transactions, blocks, state trie).

## Status

Pre-1.0. Public API may change before the 1.0 tag.

## Requirements

- PHP 8.3+

No composer dependencies.

## Credits

RLP code path lifted from [`kornrunner/ethereum-offline-raw-tx`](https://github.com/kornrunner/php-ethereum-offline-raw-tx). Format specification: <https://ethereum.org/en/developers/docs/data-structures-and-encoding/rlp/>.
